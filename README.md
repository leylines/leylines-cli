# leylines-cli

```
Usage: leylines-cli [options] [command]

CLI for leylines-geodesy

Options:
  -V, --version   output the version number
  -h, --help      display help for command

Commands:
  circle          circle geometry
  platonic        platonic geometry
  greatcircle     greatcircle geometry
  help [command]  display help for command
```
